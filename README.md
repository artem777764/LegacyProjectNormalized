# Проект
Проект является распределённым монолитом.

Бэкенд написан на ASP.NET Core, в качестве баз данных используются PostgreSQL и Redis. Раз в фиксированное время бэкенд опрашивает внешние апи и сохраняет всё в базу данных. Также он генерирует отчёты и предоставляет необходимые для фронтенда эндпоинты.

Фронтенд написан на Vue3 + Tailwind + Typescript. Фронтенд реализует навигацию по всем страницам и запрашивает все необходимые для своей работы данные у бэкенда, после чего визуализирует их.

# Установка

## Клонировать репозиторий.
```sh
git clone https://github.com/artem777764/LegacyProjectNormalized
````

## Для запуска проекта в среде разработки:

Первый терминал (Фронтенд):
````sh
cd frontend
npm install
npm run dev
````
Второй терминал (Бэкенд):
````sh
cd backend
dotnet run
````

Проект будет доступен по адрессу:
````sh
http://localhost:5173/
````

## Для запуска в среде продакшена:

````sh
docker-compose up -d
````

Проект будет доступен по адрессу:
````sh
http://localhost:2302/
````
